# === WHAT IS COMMAND PROMPT? ===

# Command Prompt (cmd) is a text-based interface on Windows
# - It's a black window where you type commands instead of clicking
# - Also called "terminal", "console", or "command line"
# - Allows you to run programs and navigate your computer with text commands

# HOW TO RECOGNIZE COMMAND PROMPT:
# - Black window with white text
# - Shows something like: C:\Users\User>
# - Has a blinking cursor waiting for you to type

# 3. Press ENTER
# 4. Wait for the command to finish (cursor will return with C:\...)

# 7. Press ENTER again
# 8. Repeat for each command

# === PERSONAL ACCESS TOKEN NOT WORKING? TROUBLESHOOTING ===

# COMMON REASONS YOUR TOKEN FAILED:

# 1. TOKEN WAS COPIED INCORRECTLY
# - Make sure you copied the ENTIRE token
# - NEW tokens start with "github_pat_" (fine-grained tokens)
# - OLD tokens start with "ghp_" (classic tokens)
# - Your token starting with "github_pat" is CORRECT and newer!
# - No spaces before or after the token
# - Copy it exactly as shown (case-sensitive)

# 2. WRONG SCOPES SELECTED
# - You MUST check the "repo" checkbox when creating token
# - Without "repo" scope, you can't push to repositories

# 3. TOKEN EXPIRED
# - If you set expiration, the token may have expired
# - Check expiration date in GitHub settings

# 4. USING WRONG USERNAME
# - Make sure username is exactly: daiyyanholmes88
# - Not your email, not your display name
